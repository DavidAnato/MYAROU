# Galerie — périmètre

Trois emplacements, chacun avec son propre jeu de photos dans le dashboard :

| Emplacement | URL publique | Édition dashboard |
|-------------|--------------|-------------------|
| **Aperçu sur l'accueil** | `/` (section MY BARIKA) | **Page d'accueil** → max **4** images |
| **Page À propos** | `/about/` | **À propos** → galerie intégrée (jusqu'à **24** images) + lien vers la page complète |
| **Page dédiée** | `/galerie/` | **Galerie** → toutes les photos (jusqu'à **50**) |

Les trois galeries sont indépendantes : vous pouvez mettre des photos différentes sur l'accueil, sur À propos et sur `/galerie/`.
